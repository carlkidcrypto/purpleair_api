PurpleAirAPIHelpers module
==========================

Utility functions used throughout the PurpleAir API wrapper for common operations
such as making HTTP requests, debugging, and data validation.

Overview
--------

This module provides helper functions that are shared across the Read, Write, and Local API modules.
These utilities handle HTTP communication, logging, and other common tasks.
