PurpleAirAPIError module
========================

Custom exception classes for error handling within the PurpleAir API wrapper.

Overview
--------

This module defines specialized exception classes used to handle various error conditions
when interacting with the PurpleAir API, including authentication failures, invalid requests,
and network issues.
