Diagrams
========

This directory contains design diagrams for the PurpleAir API project.

Dependencies
------------

Install pylint (required for pyreverse):

.. code-block:: bash

   python3 -m pip install pylint

On macOS, install Graphviz:

.. code-block:: bash

   brew install Graphviz

On Ubuntu/Debian, install Graphviz:

.. code-block:: bash

   sudo apt-get install graphviz

Auto Generate UML Diagrams
---------------------------

To automatically generate UML class diagrams for the project:

.. code-block:: bash

   pyreverse -d . -o png ../purpleair_api

This will create PNG files showing the class structure and relationships in the purpleair_api package.

Generated Files
---------------

The command above typically generates:

* ``classes.png`` - Class diagram showing all classes and their relationships
* ``packages.png`` - Package dependencies diagram
