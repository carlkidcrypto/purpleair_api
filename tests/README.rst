Tests
=====

This directory contains tests for the PurpleAir API wrapper.

Setup
-----

1. Install python coverage:

   .. code-block:: bash

      python3 -m pip install coverage

2. Install mock requests:

   .. code-block:: bash

      python3 -m pip install requests-mock

3. Remove any currently installed versions of PurpleAirAPI:

   .. code-block:: bash

      python3 -m pip uninstall purpleair_api

Running Tests
-------------

To run all tests with coverage reporting:

.. code-block:: bash

   python3 -m unittest
   coverage html -d purpleair_api_coverage_reports

Or combine in one line (note: second command only runs if first succeeds):

.. code-block:: bash

   python3 -m unittest && coverage html -d purpleair_api_coverage_reports

Test Coverage
-------------

Coverage reports are generated in the ``purpleair_api_coverage_reports`` directory.
Open ``index.html`` in a web browser to view detailed coverage information.
