PurpleAir API - Go Bindings
============================

Go bindings for the PurpleAir API, using SWIG and C++ backend.

Requirements
------------

- Go 1.21 or later
- SWIG 4.0 or later
- libcurl development headers
- C++17 compatible compiler (g++ or clang++)

Building
--------

From the repository root, run Meson to generate the SWIG wrappers and build
the native library (other language bindings are disabled to keep the build
focused and avoid unnecessary dependencies):

.. code-block:: bash

   meson setup builddir -Dcsharp_bindings=false -Djavascript_bindings=false
   ninja -C builddir

To build all language bindings at once, omit the ``-D`` flags:

.. code-block:: bash

   meson setup builddir && ninja -C builddir

Then run the Go tests from the ``go/`` directory:

.. code-block:: bash

   cd go
   go test ./tests/...

Clean generated files
~~~~~~~~~~~~~~~~~~~~~

.. code-block:: bash

   ninja -C builddir -t clean

Usage
-----

After building, import the package in your Go code:

.. code-block:: go

   package main

   import (
       "fmt"
       purpleairapi "github.com/carlkidcrypto/purpleair_api/go/purpleairapi"
   )

   func main() {
       api := purpleairapi.NewPurpleAirReadAPI("your-read-key")
       data := api.Request_sensor_data(12345, "", "")
       fmt.Println(data)
   }

Notes
-----

- The base implementation is in C++ (``swig/``).
- SWIG generates the Go bindings from the C++ headers.
- CGO is required to compile this package.
- libcurl must be installed on the target system.
